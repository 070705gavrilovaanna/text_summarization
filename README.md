# Суммаризация новостных текстов с MBart и T5

Проект по автоматическому сжатию новостных текстов с использованием двух моделей: MBart (дообученная на 250k новостей) и T5 (дообученная на том же датасете). Реализована иерархическая суммаризация для обработки длинных текстов, проведено сравнение качества обеих моделей, создан Telegram-бот для удобного использования.

**Ссылка на проект:** [ссылка на Google Colab]

---

## Этапы работы

**Технологии:** Python, PyTorch, Transformers (Hugging Face), LangChain, python-telegram-bot, MBart, T5

1. **Загрузка моделей:** 
   - MBart: базовая модель `facebook/mbart-large-cc25` + дообученные веса Ильи Гусева (`IlyaGusev/mbart_ru_sum_gazeta`)
   - T5: готовая модель `IlyaGusev/rut5_base_sum_gazeta`

2. **Суммаризация одного текста:** функция `summarize()` с настройкой параметров генерации (num_beams=5, repetition_penalty=1.4, no_repeat_ngram_size=3)

3. **Иерархическая суммаризация:** функция `hierarchical_summarize()` разбивает длинный текст на чанки (800 символов, overlap 100), суммаризирует каждый, затем объединяет результаты

4. **Постобработка:** удаление дублирующихся предложений через `clean_summary()`

5. **Тестирование:** сравнение обеих моделей на 5 текстах разных жанров (политика, юриспруденция, наука, ИИ, спорт)

6. **Telegram-бот:** интерактивный интерфейс для пользователей

## Архитектура иерархической суммаризации

```
Длинный текст (>200 символов) -> Разбиение на чанки (RecursiveCharacterTextSplitter) -> Для каждого чанка: суммаризация ->
-> Объединение суммаризаций -> Если объединенный текст >400 символов: Финальная суммаризация -> Результат
```

## Что я узнала

1. MBart превосходит T5 по точности: MBart сохраняет ключевые факты и цифры, T5 склонен к галлюцинациям

2. Иерархическая суммаризация необходима для длинных текстов: без разбиения на чанки модель обрезает текст на 1024 токена (~4000 символов)

3. repetition_penalty критичен для T5: увеличение параметра до 1.4 помогло уменьшить повторы, но не исправило галлюцинации

4. Юридические тексты - слабое место T5: модель не справляется с формальным стилем и юридической терминологией

5. MBart работает на любых жанрах: политика, юриспруденция, наука, спорт — стабильно высокое качество

## Ключевые выводы

- Выбираю MBart: надёжность, точность фактов, стабильность на любых текстах
- T5 можно использовать только для новостей и если критична максимальная степень сжатия
- Иерархическая суммаризация обязательна для текстов >4000 символов
- Параметры генерации (num_beams, repetition_penalty) сильно влияют на качество
